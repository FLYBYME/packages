# isomorphic-ui

The reactive UI framework for the isomorphic mesh ecosystem.

## Features
- **Magic Sauce Auto-Subscription**: Components automatically react to state changes.
- **Atomic Primitives**: Box, Stack, Typography, and Button.
- **Refined Patching Engine**: Targeted DOM updates.
- **Design System**: Standardized CSS tokens.

## Installation
```bash
npm install isomorphic-ui
```

See the `src/core/BrokerComponent.ts` for the core rendering logic.
